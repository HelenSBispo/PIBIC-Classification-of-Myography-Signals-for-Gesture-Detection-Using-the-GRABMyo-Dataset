# Classificação de Sinais de Miografia para Detecção de Gestos usando o Dataset GRABMyo

Este repositório contém um projeto de **classificação de sinais de miografia (sEMG)** para **detecção de gestos da mão**, utilizando o **dataset GRABMyo**.

O projeto faz parte de uma **Iniciação Científica (PIBIC/COPES)** da **Universidade Federal de Sergipe (UFS)**.

## Dataset

Os dados utilizados neste projeto são do **GRABMyo Dataset**, disponível no PhysioNet:

https://physionet.org/content/grabmyo/

O dataset contém sinais de **eletromiografia de superfície (sEMG)** coletados para estudos de **reconhecimento de gestos da mão**.

## Objetivo

O objetivo do projeto é desenvolver modelos computacionais capazes de **classificar gestos da mão a partir de sinais miográficos**, contribuindo para pesquisas em áreas como:

- reconhecimento de padrões  
- interfaces humano-computador  
- controle de próteses  

## Instituição

Projeto desenvolvido na **Universidade Federal de Sergipe (UFS)** no âmbito do programa **PIBIC/COPES**.

## Observação

Este repositório está **sujeito a atualizações contínuas**, conforme o andamento e evolução da pesquisa.
